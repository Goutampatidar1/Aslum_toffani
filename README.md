## Setup Project

1. First you have Python 3.10 Version is required
2. Install CUDA Version 12.1 From the Nvidia Official Website
3. Create Virtual environment 
```
python -m venv <your-env-name>
``` 
4. Install pytorch Using on official website for CUDA version 12.1 pip command 
5. after Doing above install requirements.txt file 
6. HAPPY RUN OUR PROJECT
